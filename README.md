# Terraform AWS: Cloud Custodian

## Requirements 
Terraform >= 1.0.5
Python >= 3.6

## Getting started 

To install Cloud Custodian CLI, you can download it using Pip:

```shell
python3 -m venv venv
source venv/bin/activate
pip install c7n 
```
* Pip package notes
  * c7n is for AWS
  * c7n_azure us for Azure
  * c7n_gcp is for Google Compute
